I just shipped a feature I've been wanting to build for weeks: personalized coaching insights for Clarity Coach, the AI communication tool I built for ESL professionals.

Here's what changed:

Clarity Coach started simple — paste a message, get four rewrites (Professional, Friendly, Concise, Executive) plus a clarity/politeness/professionalism score. Useful, but it only ever looked at one message at a time.

This week I added:
→ Accounts and history, so every message you improve gets saved
→ A progress dashboard with score trends over time
→ An AI-generated "Coach's Insight" that looks across your last several messages to catch patterns a single-message tip would miss — like consistently under-explaining requests, or a tone that reads more curt than intended

That last piece is the one I'm most proud of. It's the difference between a tool that corrects you and one that actually coaches you.

Built on Azure OpenAI, React, Node/Express, and Supabase (Postgres + auth, with row-level security so every user's history stays their own).

Eight years in the Army and years of DOD contracting across multinational teams taught me how much workplace communication struggles quietly cost people — not because they don't know their job, but because English isn't their first language and nobody built a tool that actually coaches instead of just correcting.

This is exactly the kind of work I want to keep doing: pairing Microsoft 365, Power Platform, and AI tools like Claude to build things that solve real problems, not demos.
